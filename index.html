2.1 用引用操纵对象

  • 一切都被视为对象

  • 不用new来创建变量，而是穿件一个并非是引用的”自动“变量。这个变量直接存储”值“，并置于堆栈中，因此更加高效。

2.4.1 字段与方法

  • 一旦定义了一个类（在Java中你所做的全部工作就是定义类，产生哪些类的对象，以及发送消息给这些对象），就可以在类中设置两种类型的元素：字段（有时被称为数据成员）和方法（有时被称作成员函数）

  • 基本成员默认值

    • 若类中某个成员是基本数据类型，即使没有进行初始化，Java也会确保它获得一个默认值。

DRG:Dual Relation Graph for Human-Object Interaction Detection

The task of human-object interaction detection aims to localize persons, object instances, as well as recognize the interactions (if any) between each pair of a person and an object.

Steps

(1)we first apply an off-the-shelf object detector Faster R-CNN to detect all the human/object instances in an image,Given the detected bounding boxes $b_h and b_o$, we first extract the ROI pooled features (ROI池化后的特征)and pass them into the human and object stream

(2)We then pass the detected bounding boxes as well as the object category information to the spatial-semantic stream,We apply the proposed Dual Relation Graph (DRG) in the spatial-semantic stream.

(3)Lastly, we fuse the action scores from the three streams (human, object, and spatial-semantic) to produce our final predictions

发表于 2020-07-14

SVM总结

SVM算法

1、训练流程

输入${(x_i,y_i)}_{i=1…N}$

解优化问题：

最大化：$\theta(\alpha)=\sum_{i=1}^M\alpha_i-\frac{1}{2}\sum_{i=1}^M\sum_{j=1}^M\alpha_i\alpha_jy_iy_jK(x_i,x_j)$
限制条件：1）$0<\alpha_i<=C$
2）$\sum_{i=1}^M\alpha_iy_i=0$

算$b$:

由1、我们得知所有$\alpha_i$，取一个$0<\alpha_i<=C$
$b=\frac{1-y_i\sum_{j=1}^M\alpha_jy_jK(x_i,x_j)}{y_i}$

2、测试流程

输入测试样本$X$

若$\sum_{i=1}^M\alpha_iy_iK(x_i,X)+b>=0$,则$y=+1$
若$\sum_{i=1}^M\alpha_iy_iK(x_i,X)+b<0$,则$y=-1$

SVM_nolinear

发表于 2020-07-09

SVM处理非线性问题

1、优化问题

最小化：$\frac{1}{2}||\omega||^2+C\sum_{i=1}^N\xi _i$
限制条件:a、$y_i[\omega ^Tx_i+b]>=1-\xi$ ;b、$\xi _i>=0 (i=1...N)$

2、低维到高维映射$\phi(x)$

$x \rightarrow \phi(x)$
$\Rightarrow y_i [\omega ^Tx_i+b]>=1-\xi \rightarrow y_i[\omega ^T \phi(x_i)+b]>=1-\xi$

注意：$\phi(x)$是无限维，从而$\omega$也将是无限维的

$\Rightarrow$

定理：我们可以不知道无限维映射$\phi(x)$的显式表达，我们只要知道一个核函数(Kernel Function):$K(x_i,x_j)=\phi(x_i)^T\phi(x_j)$，则优化问题（1）仍然可解。

3、原问题和对偶问题

1）原问题(Prime Problem)

最小化：$f(\omega)$
限制条件：a、$g_i(\omega)<=0 (i=1...K)$
b、$h_i(\omega)=0 (i=1...M)$

2)对偶问题(Dual Problem)

1、定义：$L(\omega,\alpha,\beta)$
$=f(\omega)+\sum_{i=1}^K \alpha_i g_i(\omega)+\sum_{i=1}^M \beta_i h_i(\omega)$
$=f(\omega)+\alpha^T g(\omega)+\beta^T h(\omega)$
2、对偶问题定义：
最大化：$\theta(\alpha,\beta)=inf{L(\omega,\alpha,\beta)}$
限制条件：$\alpha_i>=0 (i=1...K)$

定义：如果$\omega^*$是原问题的解，而$\alpha^*,\beta^*$是对偶问题的解，则有：

$f(\omega^*)>=\theta(\alpha^*,\beta^*)$
证：$\theta(\alpha^*,\beta^*)=inf{L(\omega,\alpha^*,\beta^*)}<=inf{L(\omega^*,\alpha^*,\beta^*)}=f(\omega^*)+\sum_{i=1}^K \alpha_i^* g_i(\omega^*)+\sum_{i=1}^M \beta_i^* h_i(\omega^*)<=f(\omega^*)$

定义：$G=f(\omega^)-\theta(\alpha^\,\beta^*)>=0，G$叫做原问题与对偶问题的间距(Duality Gap)

$\Rightarrow$

强对偶定理：若$f(\omega)$为凸函数，且$g(\omega)=a\omega +b,h(\omega)=c\omega+d,$则由此优化问题的原问题与对偶问题的间距（$G$）为0，即$f(\omega^*)=\theta(\alpha^*,\beta^*)$

$\Rightarrow (KTT条件)$ 对$\forall i=1…K，$或者$\alpha _i^*=0$，或者$g_i^*(\omega ^*)=0$。

一、没有免费午餐定理（No Free Lunch Theorem）

​ 如果我们不对特征空间有先验假设，则所有算法的表现是一样的。

e.g.

如果不假定预测为哪一类的概率更大，则出现各类的概率一样，那么，预测为“⭕️”获“❌”都合理。

二、支持向量机（Support Vector Machine）

定义：

1、训练数据及标签$(x_1,y_1),(x_2,y_2)…(x_N,y_N)$。$x_i$为列向量，$y_i$为标签取值为+1或-1。

2、线性模型：

$(\omega,b)$
$\omega ^T+b=0$（超平面）

其中$\omega$为列向量，$b$为常数。

3、一个训练集线性可分是指：${(x_i,y_i)}_{i=1…N},\exists (\omega,b)$使得对$\forall i=1…N$有

a、若$y_i=+1$则$\omega ^Tx_i+b>=0$
b、若$y_i=-1$则$\omega ^Tx_i+b<0$
$\Rightarrow y_i[\omega^Tx_i+b]>=0$
